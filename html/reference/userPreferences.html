User Preferences

Parameter Description
Import convertible fares only The import of 90918-10 data is limited to fares marked for conversion.
Import only stations from listed countries The import of MERITS stations is limited to countries listed.
Ignore optional via stations Optional via stations are ignored in the import of 108.1 data.
