Converter version 2.1.15

New Features:

- export of TCVC table (carrier code and names of the involved carriers from the UIC RICS table)
- new indication of test data delivery in the OSDM delivery data. Data deliveries marked as test data can not be uploaded to the production platform.

other changes:

- return prices filled with twice the single price in fare tables (in conversion to 108).
- new error message in case of missing names for start or end stations (in conversion to 108).
