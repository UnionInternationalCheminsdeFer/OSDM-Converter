Company codes

    Company codes identify a carrier. The official company codes must be used. The reference data file can be downloaded from the UIC web site.

    Company codes can be imported from a UTF-8 encoded csv file with the following columns:

columndatadescription
1company code4 position company code
2short nameshort name of the company
3long namelong name of the company
4..other columns will be ignored. They might be included in the csv file for documentation.
