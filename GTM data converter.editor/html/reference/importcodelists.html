Code Lists

    The conversion needs reference data for stations, companies and service brands.

    Company codes can be imported via a csv file created from the UIC RICS Code List taken from the UIC web site. The csv file is assumed to be UTF-8 encoded, so the RICS code table (EXCEL) must be stored using the csv UTF-8> option.

    Service Brand codes can be imported via a csv file created from the UIC Service Brand Code List taken from the UIC web site. The csv file is assumed to be ASC-II encoded, so the table (EXCEL) must be stored using the csv> option.

    Stations can be imported via the TSDUPD file from MERITS. The file is formated in EDIFACT according to the MERITS/TAP-TSI specification.

    Legacy border point datas can be imported via a csv file from the border point list provided on the UIC extranet. The same border point data are required for the conversion of border points to and from OSDM.

Reference data

    Code lists to be imported:

    • carriers - UIC company codes
    • service brands - UIC service brands
    • stations - MERITS stations
    • NUTS codes - regional codes (not used for pure conversion)
    • legacy border point codes

Import reference data

    90918-10 fare data are based on reference data for company codes, service brands and stations. These code tables need to be imported first.
    The data will be shown in the code list section of the data set.
