Conversion between OSDM 1.2 and OSDM Version 1.4

    The Export of data to the OSDM format supports the version 1.2 and 1.4. The version can be selected in the Delivery data.

    The import of OSDM data supports the versions 1.2 and 1.4. The internal representations is aligned to OSDM version 1.4.

    A conversion can be done via an Import of OSDM data, a change of the schema version in the delivery and an export of the data.

    In the export to version 1.2 only the first carrier in a carrier restriction of the route will be included in the exported data.
